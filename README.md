# Apartment Price Prediction — Poland

A machine learning project to predict apartment prices in Poland using regression techniques.

## Overview

This project analyzes real estate data from Polish cities to build a predictive model for apartment prices. It explores the relationship between various features (location, size, number of rooms, etc.) and the final price per square meter.

## Features

- Exploratory Data Analysis (EDA) with visualizations
- Data cleaning and preprocessing
- Feature engineering and selection
- Multiple regression models compared (Linear Regression, Random Forest, etc.)
- Model evaluation with RMSE and R² metrics

## Tech Stack

- Python
- Jupyter Notebook
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Results

The best performing model achieved accurate predictions of apartment prices based on key features such as location, surface area, and number of rooms.
